Help the operator write a warm thank-you note to a donor.

Ask for:
- The donor's name
- What they gave (a dollar amount, or an in-kind gift / their time)
- Anything personal worth mentioning (a longtime supporter, a specific program they care about, etc.)

Then write a short, warm, plain-English thank-you in LCAC's voice — community-focused, never corporate (see the brand tone in CLAUDE.md). Use words like family, community, support, belong; avoid "stakeholders," "leverage," etc.

IMPORTANT — this command does NOT send any email. It only writes the words. Output the draft as plain text so the operator can read it, tweak it, and send it from her own inbox. Nothing is automated and nothing costs money.
